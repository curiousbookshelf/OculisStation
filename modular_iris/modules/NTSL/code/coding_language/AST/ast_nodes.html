/datum/nodeAn abstract syntax tree (AST) is a representation of source code in a computer-friendly format. It is composed of nodes, each of which represents a certain part of the source code. For example, an node represents an if statement in the script's source code. Because it is a representation of the source code in memory, it is independent of any specific scripting language. This allows a script in any language for which a parser exists to be run by the interpreter.
